# Makiel
Web scrapper for [abidanarchive.com](https://abidanarchive.com)

## Installation / Use
Have [node](https://nodejs.org/en/) installed.
Administrator account credentials optional. Won't get everything without them.

```
npm ci
node makiel [ADMIN USERNAME] [ADMIN PASSWORD]
cd reports
```

Honestly I'm probably the only person who's going to use this so not sure why I'm writing this.
